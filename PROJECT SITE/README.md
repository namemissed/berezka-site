# Берёзка

Проект переведён на PHP + MySQL и подготовлен под обычный хостинг с поддержкой PHP 8+.

## Структура

- `index.php` — публичная главная страница (базовая версия)
- `pages/index_full.php` — полная версия страницы с сотрудниками и методиками
- `admin/login.php` — вход в админ-панель
- `admin/index.php` — редактирование контента сайта (3 вкладки: Главная, Сотрудники, Методики)
- `includes/` — подключение к БД, авторизация и работа с данными
- `config/config.php` — настройки подключения к базе
- `database/schema.sql` — SQL-схема и стартовые данные
- `assets/css/` — стили сайта и админ-панели
- `uploads/staff/` — папка для фотографий сотрудников

## Что нужно для запуска на хостинге

1. Хостинг с PHP 8+ и MySQL.
2. Создать базу данных.
3. Импортировать `database/schema.sql`.
4. В `config/config.php` указать данные базы.
5. Заменить в `database/schema.sql` строку `REPLACE_WITH_PASSWORD_HASH_FROM_PASSWORD_HASH_FUNCTION` на реальный хеш пароля.
6. Загрузить файлы проекта в корень сайта.

## Как получить хеш пароля

На любом хостинге с PHP выполни:

```php
<?php
echo password_hash('admin12345', PASSWORD_DEFAULT);
```

Полученную строку вставь вместо `REPLACE_WITH_PASSWORD_HASH_FROM_PASSWORD_HASH_FUNCTION`, затем импортируй SQL или обнови запись администратора.

## Что можно редактировать через админ-панель

### Вкладка "Главная"
- тексты первого экрана;
- подписи и ссылки кнопок;
- карточки направлений;
- блок преимуществ;
- текст в подвале.

### Вкладка "Сотрудники"
- ФИО сотрудников;
- должности;
- описание (образование, опыт);
- имена файлов фотографий.

### Вкладка "Методики"
- названия методик;
- краткое описание;
- подробное описание.

## Таблицы базы данных

- `site_settings` — настройки сайта (тексты, заголовки)
- `programs` — карточки направлений деятельности
- `highlights` — факты и преимущества
- `staff` — сотрудники детского сада
- `methodologies` — методики работы
- `admins` — учётные записи администраторов

## Следующий шаг

Можно расширить функционал:

- загрузкой изображений через админ-панель;
