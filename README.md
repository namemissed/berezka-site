# Берёзка

Сайт подготовлен для публикации на GitHub Pages и редактирования через Pages CMS.

## Что редактируется

- `_data/site.yml` — основные тексты и кнопки
- `_data/programs.yml` — карточки направлений
- `_data/highlights.yml` — факты и преимущества

## Как это работает

- `index.html` — шаблон страницы
- `styles.css` — оформление
- `.pages.yml` — конфигурация Pages CMS
- GitHub Pages собирает сайт автоматически
